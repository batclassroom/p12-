HTML Responsive

HTML Responsive c'est quoi ?

HTML Responsive désigne la pratique de créer des sites web qui s'adaptent automatiquement à différentes tailles d'écran et dispositifs, tels que les smartphones, tablettes, et ordinateurs de bureau. L'objectif est de fournir une expérience utilisateur optimale quel que soit le dispositif utilisé.

Dans ce projet

J'ai commencé par développer la version mobile du site ensuite la version tablette et enfin la version desktop.
