Mon SCSS

SCSS c'est quoi ?

SCSS (Sassy CSS) est une des deux syntaxes disponibles pour le préprocesseur CSS Sass (Syntactically Awesome Style Sheets). SCSS apporte des fonctionnalités avancées au CSS, ce qui le rend plus puissant et maintenable.

Dans ce projet

J'ai découvert l'écriture de SCSS, qui au début me paraissait plutôt compliquer à comprendre car le code est réduit. Mais en utilisant les mixins j'ai alors compris l'avantage de cette écriture qui permet de soulager grandement le code CSS qui est parfois très long.
