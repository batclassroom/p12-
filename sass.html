Mon SASS

SASS c'est quoi ?

Sass (Syntactically Awesome Style Sheets) est un préprocesseur CSS qui ajoute des fonctionnalités et des capacités supplémentaires aux feuilles de style traditionnelles. Sass permet d'écrire du CSS plus rapidement et plus efficacement grâce à des fonctionnalités avancées comme les variables, les imbriquations, les mixins, les fonctions et l'héritage.

Dans ce projet

J'ai découvert l'écriture de SASS, qui au début me paraissait plutôt compliquer à comprendre car le code est réduit. Mais en utilisant les mixins j'ai alors compris l'avantage de cette écriture qui permet de soulager grandement le code CSS qui est parfois très long.
